Installing the App Store

When you click on Install R2 you'll be taken to Salesforce.com Sandbox Login, you'll need to login with you Sandbox Credentials

Salesforce Connected Apps we'll show you several screens asking for permission to enable R2 for your SANDBOX users.

After that your users will be able to login with salesforce into your own App Store

The first time each users log's in the Connected App Permission Box will appear in order to Allow Access.

Creating the App Store

In order to setup your Dev Environment you need NODEJS and GIT

Clone the App Store from github

Install all dependencies with $ npm install .

Edit ./config/organization.json and add your Salesforce.com ORG_ID

On the R2's root path in terminal run $ grunt server

R2's is running on the Web Browser at localhost:7770
