Features



1

Front End as a Service (FaaS)


Build an App for That!

3VOT is a Front-End as a Service to build Apps 5X Faster with reusable NPM components and the best of the modern collaborative models.

3VOT provides you with a Profile for your organization, watch this video to see how it works.

With 3VOT you don't have to worry about servers, infraestruture or other complex decisions

2




Just popular Javascript, HTML and CSS code needed.

Reusable Components Architecture inspired in Github and Node Packaged Modules.

Integrate any NPM component into your code or even clone an existing 3VOT App as a starting base you can extend or modify.

3




API-hub to easily connect to third party API services, ideal for consuming data from services like Parse.com in your apps.

API-proxy to instantly connect data from Salesforce.com to your apps.

Infrastructure able to jump from 1 to millions of users with confidence, performance and cost efficiency.


4




Multi-platform optimized to obtain outstanding performance in mobile browsers.

Organization's Profile to categorize apps in Stores and publish them for your users, departments, partners or customers.

App Marketplace to promote your apps to the world and 3VOT's community.
