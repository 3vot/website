Features



1

Front End as a Service (FaaS)


Build an App for That!

3VOT is a Front-End as a Service (FaaS) to build Enterprise Apps 5X Faster.

Forget about having to take hundreds of complicated decisions to start your multi-platform App development. No complex cloud servers to manage, no risky software architecture decissions, no coding rockstars needed to start.

3vot sets it all for you so you can start building your app immediately. You can even start clonning an existing 3VOT App to modify or extend it!

2




Just popular Javascript, HTML and CSS code needed.

Reusable Components Architecture inspired in Github and Node Packaged Modules.

Integrate any NPM component into your code or even clone an existing 3VOT app as a starting base you can extend or modify.

3




API-hub to easily connect to third party API services and Back-ends. Manage sessions, tokens and security. Ideal for consuming data from services like Parse.com in your apps.

Take advantage of 3VOT-R2 an API-proxy to connect data from Salesforce.com to your apps using REST standards, large query pagination and security management.

Edge Front-End infrastructure able to jump from 1 to millions of users with confidence, performance and cost efficiency.


4




Multi-platform optimized to obtain outstanding performance in mobile browsers.

Organization's Profile to publish 3VOT Apps developed by internal or citizen developers. Categorize apps in Stores to publish them for your users, departments, partners or customers.

App Marketplace to clone or share free apps with 3VOT's community and buy or sell private apps.
